# Décor.AI — Interior Design Studio

AI-powered interior design app. Upload a room photo and get two redesigned versions (Elegant + Casual) generated by Claude + DALL-E 3.

---

## 🚀 Deploy to Vercel (step-by-step)

### Step 1 — Upload to GitHub
1. Go to [github.com](https://github.com) and create a new repository called `decor-ai`
2. Upload all these files by dragging them into the GitHub file uploader
3. Click **Commit changes**

### Step 2 — Deploy on Vercel
1. Go to [vercel.com](https://vercel.com) → **Add New Project**
2. Click **Import** next to your `decor-ai` GitHub repo
3. Vercel auto-detects Next.js — no config needed
4. Before clicking Deploy, go to **Environment Variables** and add:
   - `ANTHROPIC_API_KEY` = your key from [console.anthropic.com](https://console.anthropic.com)
5. Click **Deploy**
6. In ~60 seconds you get a live URL like `https://decor-ai.vercel.app`

### Step 3 — Use the app
1. Open your Vercel URL
2. Enter your OpenAI API key in the banner (saved to your browser — only once!)
3. Upload a room photo and click **Generate Design Concepts**

---

## 🔑 API Keys needed

| Key | Where to get it | Where it lives |
|-----|----------------|----------------|
| Anthropic (Claude) | [console.anthropic.com](https://console.anthropic.com) | Vercel environment variable (secure) |
| OpenAI (DALL-E 3) | [platform.openai.com/api-keys](https://platform.openai.com/api-keys) | Your browser localStorage |

## 💻 Run locally (optional)

```bash
npm install
cp .env.example .env.local
# Edit .env.local and add your ANTHROPIC_API_KEY
npm run dev
```
Open [http://localhost:3000](http://localhost:3000)
